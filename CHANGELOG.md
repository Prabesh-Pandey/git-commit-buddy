# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-02-18

### Added

- Added `CONTRIBUTING.md` with development guidelines.
- Added `.editorconfig` for consistent formatting.
- Added Troubleshooting section to README.
- Added support for `Pipfile.lock` and `yarn.lock` in commit message templates.
- Added more ignore patterns to `.gitignore` (logs, OS files).
- Added error logging to `getCurrentBranch` helper.

### Fixed

- Fixed repository URL in `package.json`.
- Fixed VS Code version requirement in README to match engine constraint.

### Changed

- Improved `.gitignore` structure.

### Removed

- Removed stale test comments in `ui.js`.

## [1.0.0] - 2026-01-27

### Added

- Initial release of Git AutoPush on Save.
- AI-powered commit messages via OpenRouter.
- Gamification with streaks and achievements.
- Status bar integration and stats dashboard.
